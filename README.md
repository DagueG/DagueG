## À propos

Je développe des **systèmes d'IA et d'automatisation en production**. Spécialisé dans les pipelines ML robustes, les APIs scalables et l'optimisation basée sur les données. Pour des raisons de confidentialité, les projets d'automatisation complets réalisés pour des entreprises ne seront pas publiés sur ce GitHub.

**Focus actuel :** Systèmes RAG, MLOps, Prédiction & Optimisation métier

---

## Projets Principaux

### RAG – Système de recommandation d'événements culturels

Système RAG complet pour recommander des événements culturels. Pipeline production avec API OpenAgenda, embeddings, indexation Faiss et LLM Mistral.

**Tech:** `Python` `LangChain` `Mistral AI` `Faiss` `FastAPI` `Docker`

[Voir le repo →](https://github.com/DagueG/RAG)

---

### Model Machine Learning – API Prédiction Énergétique

API REST pour prédire la consommation énergétique des bâtiments. Architecture modulaire avec validation Pydantic, PostgreSQL et tests automatisés.

**Tech:** `FastAPI` `PostgreSQL` `SQLAlchemy` `Scikit-learn` `Pydantic` `Docker`

[Voir le repo →](https://github.com/DagueG/Model_Machine_Learning)

---

### MLOps – Scoring de Crédit avec Experimentation

Pipeline ML avancé pour scoring de crédit. Gestion du déséquilibre de classe, optimisation d'hyperparamètres, tracking MLflow et explainability SHAP.

**Tech:** `XGBoost` `LightGBM` `MLflow` `SHAP` `Scikit-learn` `Jupyter`

[Voir le repo →](https://github.com/DagueG/MLOps)

---

### Building Consumption – Data Science Énergétique

Analyse complète d'un dataset réel (NYC 2016) : exploration, feature engineering et modélisation pour prédiction de consommation énergétique.

**Tech:** `Jupyter` `Pandas` `Scikit-learn` `Matplotlib`

[Voir le repo →](https://github.com/DagueG/building_conso_data_predict)
